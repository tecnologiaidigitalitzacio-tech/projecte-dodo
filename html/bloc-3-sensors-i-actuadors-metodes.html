Mètodes de la Classe Dodo

En aquest notebook, explorarem els mètodes de la classe dodo de la llibreria libreria-dodo. Aquesta llibreria s'utilitza per controlar el robot Dodo, permetent gestionar motors, LEDs RGB i altres components.

Creació de l'objecte dodo

from complubot.libreria_dodo import dodo

# Inicialitzar l'objecte del robot
mi_robot = dodo()

mueve_motor(motor, velocidad)

Aquest mètode permet controlar els motors del robot.

  • motor: Número del motor (1 a 4 per a Dodo, 1 a 2 per a Dodo Lite).
  • velocidad: Percentatge de velocitat (-100 a 100).
# Exemple: Mou el motor 1 al 80%
mi_robot.mueve_motor(1, 80)

para_motor(motor)

Aquest mètode atura un motor específic.

  • motor: Número del motor a aturar.
# Exemple: Para el motor 1
mi_robot.para_motor(1)

enciende_rgb(pos, r, g, b)

Aquest mètode encén un LED RGB en una posició específica amb un color determinat.

  • pos: Número del LED (1 a 5).
  • r, g, b: Intensitat de cada color (0 a 255).
# Exemple: Encén el LED RGB 1 en vermell
mi_robot.enciende_rgb(1, 255, 0, 0)

apaga_rgb(pos)

Aquest mètode apaga un LED RGB específic.

  • pos: Número del LED a apagar (1 a 5).
# Exemple: Apaga el LED RGB 1
mi_robot.apaga_rgb(1)

enciende_buzzer(frecuencia)

Aquest mètode encén el buzzer amb una freqüència específica.

  • frecuencia: Freqüència en Hz.
# Exemple: Encén el buzzer a 440 Hz (nota La)
mi_robot.enciende_buzzer(440)

apaga_buzzer()

Aquest mètode apaga el buzzer.

# Exemple: Apaga el buzzer
mi_robot.apaga_buzzer()

lectura_s1()

Aquest mètode llegeix el sensor S1.

# Exemple: Llegeix el sensor S1
valor_s1 = mi_robot.lectura_s1()
print(valor_s1)

lectura_s2()

Aquest mètode llegeix el sensor S2.

# Exemple: Llegeix el sensor S2
valor_s2 = mi_robot.lectura_s2()
print(valor_s2)

lectura_s3()

Aquest mètode llegeix el sensor S3.

# Exemple: Llegeix el sensor S3
valor_s3 = mi_robot.lectura_s3()
print(valor_s3)

lectura_s4()

Aquest mètode llegeix el sensor S4.

# Exemple: Llegeix el sensor S4
valor_s4 = mi_robot.lectura_s4()
print(valor_s4)

lectura_bateria()

Aquest mètode retorna el nivell de bateria del robot.

# Exemple: Llegeix el nivell de bateria
bateria = mi_robot.lectura_bateria()
print(bateria)

detectar_obstaculo()

Aquest mètode detecta si hi ha un obstacle davant del robot.

# Exemple: Detecta obstacle
obstacle = mi_robot.detectar_obstaculo()
print(obstacle)
